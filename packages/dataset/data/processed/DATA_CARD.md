# BBB Peptide Dataset Card

## Summary

- This file is generated by `tfg_bbb.data_card.render_data_card`.
- Rebuild the dataset through `tfg-bbb-build` to refresh row counts and source statistics.

## Pipeline

- Source loaders: `tfg_bbb.sources`
- Cleaning and deduplication: `tfg_bbb.clean`
- Feature extraction: `tfg_bbb.features`
- Splits: `tfg_bbb.splits`
- Export and EDA: `tfg_bbb.pipeline`

## Defaults

- Length bounds: `6-30`
- Identity threshold: `0.9`
- Fold strategy: `StratifiedGroupKFold` on `cluster_id`

## Notes

- UniProt/PDB cross-references are intentionally excluded.
