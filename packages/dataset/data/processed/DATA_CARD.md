# BBB Peptide Dataset Card

## Summary

- This file is generated by `bbb_dataset.data_card.render_data_card`.
- Rebuild the dataset through `bbb-dataset-build` to refresh row counts and source statistics.

## Pipeline

- Source loaders: `bbb_dataset.sources`
- Cleaning and deduplication: `bbb_dataset.cleaning`
- Feature extraction: `bbb_dataset.features`
- Splits: `bbb_dataset.splits`
- Export and EDA: `bbb_dataset.builder`

## Defaults

- Length bounds: `6-30`
- Identity threshold: `0.9`
- Fold strategy: `StratifiedGroupKFold` on `cluster_id`

## Notes

- UniProt/PDB cross-references are intentionally excluded.
